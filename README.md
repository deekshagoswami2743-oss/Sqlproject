# 🍕 Pizza Sales Analysis (SQL Project)

## 📌 Project Overview
This project analyzes a pizza sales dataset using SQL to uncover meaningful business insights. By working with multiple related tables—`orders`, `order_details`, `pizzas`, and `pizza_types`—the project demonstrates how raw transactional data can be transformed into actionable insights through structured querying and logical analysis.

---

## 🎯 Objectives
The main goal of this project is to answer key business questions across different levels of complexity:
- Understand overall sales performance
- Identify customer preferences
- Analyze product and category trends
- Evaluate revenue contribution and growth patterns

---

## 🛠️ Tools & Technologies
- SQL (MySQL/PostgreSQL/SQLite)
- Relational Database Concepts

---

## 🗂️ Dataset Description
The dataset consists of four tables:
- **orders** → Contains order ID, date, and time  
- **order_details** → Includes order quantity and pizza IDs  
- **pizzas** → Contains pizza size and price  
- **pizza_types** → Provides pizza names and categories  

---

## 📊 Analysis Breakdown

### 🔹 Basic Analysis
- Total number of orders placed  
- Total revenue generated  
- Highest-priced pizza  
- Most commonly ordered pizza size  
- Top 5 most ordered pizza types  

### 🔹 Intermediate Analysis
- Total quantity of pizzas ordered by category  
- Distribution of orders by hour  
- Category-wise pizza distribution  
- Average number of pizzas ordered per day  
- Top 3 pizza types based on revenue  

### 🔹 Advanced Analysis
- Percentage contribution of each pizza to total revenue  
- Cumulative revenue over time  
- Top 3 pizzas by revenue within each category (using window functions)  

---

## 💡 Key SQL Concepts Used
- **SELECT, WHERE, ORDER BY, LIMIT**
- **JOINS (INNER JOIN)**
- **GROUP BY & Aggregate Functions (COUNT, SUM, AVG)**
- **Subqueries**
- **Common Table Expressions (CTEs)**
- **Window Functions (RANK, DENSE_RANK, SUM OVER)**

## 📈 Key Insights

- Highlights peak order times and customer demand patterns  
- Reveals revenue-driving items and growth trends  
- Helps understand sales distribution across categories
